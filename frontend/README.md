# QuickBite Frontend

This directory contains the frontend applications for the QuickBite platform.

## Projects

- **[Admin Dashboard](./admin_quickbite)**: Web portal for platform administrators to manage users, restaurants, and monitor orders.
- **[Customer App](./customer_quickbite)**: Web application for customers to browse restaurants, view menus, and place orders.
- **[Delivery App](./delivery_quickbite)**: Web application for delivery partners to manage and fulfill orders.
- **[Restaurant Dashboard](./restaurant_quickbite)**: Web portal for restaurant owners to manage their menu, orders, and view analytics.

## Getting Started

Each project is a separate React application. Navigate to the respective directory and follow the instructions in its `README.md` to start the development server.
