# QuickBite Admin Dashboard

The Admin Dashboard is a comprehensive management tool for QuickBite platform administrators.

## Live URL
[https://admin-quickbite.vercel.app/](https://admin-quickbite.vercel.app/)

## Features

- **Dashboard Overview**: Real-time metrics on orders, revenue, and active users.
- **User Management**: View and manage customer and delivery partner accounts.
- **Restaurant Management**: Approve, edit, or remove restaurant listings.
- **Order Monitoring**: Track active orders and view order history.
- **Analytics**: Detailed reports on platform performance.

## Tech Stack

- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **State Management**: React Context / Hooks
- **Routing**: React Router

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
