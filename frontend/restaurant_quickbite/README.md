# QuickBite Restaurant Dashboard

The Restaurant Dashboard empowers restaurant owners to manage their presence on the QuickBite platform.

## Features

- **Order Management**: Accept, prepare, and mark orders as ready.
- **Menu Management**: Add, edit, and remove menu items and categories.
- **Business Insights**: View sales data, popular items, and customer feedback.
- **Settings**: Manage restaurant profile, opening hours, and delivery zones.

## Tech Stack

- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **State Management**: React Context / Hooks
- **Routing**: React Router

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
